The Macpi

This project was an attempt to create a modern, working computer out of a broken 1986 Apple Macintosh SE, while maintaining the original aesthetics by not modifying the outward appearance. This involved a long and tedious process of screen swapping, adapting video signals, and cable management, all of which can be viewed in the official Macpi project page! The end result was a working system, a second monitor, and a happy Quinn.

Programming Side Projects

Every so often, I am compelled to code something utterly useless for fun. In fact, this entire website originated as one of those projects. The really obscure text-based homepage was all done just to see if I could. These programs are all over the internet, but I have attempted to implement as many of them as possible into the homepage of this website. Typing will bring up a console, which can be used to run many of these projects. Highlights for me include Conway's Game of Life ('GOL') and Snake ('SNEK').

Programming Language

This programming language, currently called "language", is an ongoing attempt between two other students and I to create a programming language from scratch. Everything is based around the principles of functional programming, with a completely unambiguous syntax. View the documentation and current progress on GitHub.
